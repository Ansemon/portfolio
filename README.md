# GitHub Pages starter

Plain HTML/CSS site — no Jekyll, no build step. A `.nojekyll` file is
included so GitHub Pages serves these files exactly as-is.

## Structure
- `index.html` — landing page, links to the three pages below
- `about.html` — bio placeholder
- `projects.html` — project list placeholder
- `documents.html` — PDF download list
- `pdfs/` — put your real PDF files here
- `style.css` — shared styling

## Deploy
1. Push this to a GitHub repo (see chat instructions for the full walkthrough)
2. Settings → Pages → Source → Deploy from a branch → main → / (root)
3. Visit the URL GitHub gives you
